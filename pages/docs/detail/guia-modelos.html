Guia de Estilo e Templates

Publicado em 23 de Novembro de 2025 | Por: Eduardo

Resumo: Este documento serve como o manual de construção do Hub Digital PARADOCX. Aqui detalhamos os 6 modelos de layout (Templates) que compõem nossa arquitetura SPA, explicando quando e como usar cada um.

Por que Templates?

Para manter a consistência visual e agilidade no desenvolvimento (Custo Zero), adotamos um sistema modular. Em vez de criar cada página do zero, utilizamos "esqueletos" pré-definidos que garantem que a identidade Dark/Glassmorphism seja mantida.

"A repetição cria o padrão. O padrão cria a consistência. A consistência cria a confiança."

Catálogo de Modelos

Abaixo, a lista dos layouts disponíveis no sistema. Clique para visualizar a versão "Crua" (Raw) de cada um.

1. Layout Principais (Landing)

Usado para as páginas "Pilar" do site (Projetos, Conexões). Focado em causar impacto visual com Grids de Cards e pouca leitura densa.

  • Uso: Vitrines, Hubs de Navegação.
  • Ver Modelo Ao Vivo ↗

2. Layout Docs (Leitura)

O modelo que você está lendo agora. Focado em tipografia, hierarquia e retenção de atenção. Possui Sidebar técnica e Bio do Autor.

  • Uso: Artigos, Notícias, Detalhes de Projetos.
  • Ver Modelo Ao Vivo ↗

3. Layout Pesquisa (Data-Driven)

Estrutura otimizada para filtrar grandes quantidades de dados (Glossário, FAQ, Lista de Docs). Possui campo de busca centralizado.

  • Uso: Bibliotecas, FAQs, Glossários.
  • Ver Modelo Ao Vivo ↗

4. Layout Estático (Institucional)

Para conteúdo denso que não precisa de muita firula visual, mas exige sobriedade. Sidebar de navegação interna (âncoras) é o destaque.

  • Uso: Termos de Uso, Sobre, Cronologia.
  • Ver Modelo Ao Vivo ↗

5. Layout Formulários (Conversão)

Design "Split-Screen" (Tela Dividida). Imagem inspiracional à esquerda, formulário focado à direita.

  • Uso: Junte-se, Login, Contato Específico.
  • Ver Modelo Ao Vivo ↗

6. Layout Personalizável (Livre)

Uma tela em branco. O CSS não interfere, permitindo criatividade total (ex: Faixa Azul dos Integrantes, Carrossel da Home).

  • Uso: Home, Páginas Especiais, Perfis.
  • Ver Modelo Ao Vivo ↗
